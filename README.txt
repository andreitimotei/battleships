Programul porneste prin comanda ./battleship harta.txt.
Se deschide meniul, de unde jucatorul poate incepe un joc nou
prin apasarea new game, si isi poate configura asezarea navelor
prin apasarea configure map. Configurarea realizata aici, daca
e valida, va fi salvata in fisierul harta.txt si va fi folosita
implicit, pana la o eventuala modificare (tot din sectiunea 
configure map). In joc, atat selectarea spatiilor, cat si a
optiunii "destroy in advance", se face prin Enter. Deplasarea
se face cu ajutorul sagetilor. Din fiecare meniu se iese prin
apasarea butonului 'q'. Din joc se iese cu 'quit'. In timpul
jocului, la distrugerea unei nave a adversarului, pe ecran 
apare o instiintare ca ati distrus o nava, iar aceasta se co-
loreaza in galben. 

Bonus (nu sunt sigur ca se incadreaza): optiunea de configure
map, de unde jucatorul isi poate aranja navele in-game.
